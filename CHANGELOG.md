## [0.0.1] - 2026-01-26

### Added

- 初始化 Vue3 + Vite + Element Plus 前端框架
- 完成基础登录页面与路由系统
- 集成基础 Layout 布局框架
- 初始化 Node.js + Express 后端服务
- 实现登录接口与 JWT 鉴权逻辑
- 提供 MySQL 数据库初始化 SQL
- 提供前后端本地开发一键启动脚本
- 提供 Linux 服务器 deploy.sh 一键部署脚本
- 提供完整本地与线上部署教程文档

### Changed

- Layout 结构参考 vue-vben-admin 实现，后续将逐步优化简化

### Fixed

- 修复首次部署数据库权限与初始化问题
- 修复服务器端 Node 依赖安装缺失问题
